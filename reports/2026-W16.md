# Archived — Breast Cancer Sample Report (Original Fork)

This file was the sample report from the original `breast-cancer-uptodate` fork.
It has been superseded by the hematology dual-mode system.

New reports are generated as:
- `reports/malignant-YYYY-WNN.md` — hematological malignancies
- `reports/benign-YYYY-WNN.md` — non-malignant hematology

Run: `uv run python main.py run -m malignant` or `uv run python main.py run -m benign`
